# Uma Traders - AI-Powered Tiles Visualizer

An industry-grade tiles visualization platform featuring automatic AI floor segmentation, manual paint/refine tools, perspective warping, and integrated subscription billing.

## Features
- **Neural Floor Segmentation:** Automatic scanning powered by Segformer B5.
- **Precision Refinement:** Photoshop-style manual brush, eraser, and Smart Fill (Magic Wand) flood-fill.
- **Warp & Blend:** Real-time perspective warp with exposure, shadow, and softness controls.
- **Razorpay Billing:** Integrated premium plan tiers and secure online checkout.
